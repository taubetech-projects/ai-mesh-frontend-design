# AI Mesh Frontend (Next.js)

## ESLint, TypeCheck and Build

**ESLint** checks code style, **TypeCheck** ensures type correctness, and **Build** compiles the app and fails on critical errors.

```bash
npm run lint
npm run typecheck
npm run build
```

## Run

```bash
npm install
npm run dev
```
